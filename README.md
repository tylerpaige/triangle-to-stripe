# triangles into stripes

[see it live](https://triangle-to-stripe.netlify.app)

The purpose of this project is to make shapes fall from the ceiing, flatten as others lay on top of them, and ultimately turn into stripes. Like many of my projects, it's ultimately a means to deliver text -- although in this case I am still working on a script I want to put forward. I imagine I will try out many versions of this over the years, so for now I'm interested in getting a format that feels right.

### Controls

- scroll to control the animation playback
- press and hold to auto-play. The bottom half moves forward, the top half reverses
- Double-tap to skip 10%. again, the bottom half moves forward and the to half moves in reference
